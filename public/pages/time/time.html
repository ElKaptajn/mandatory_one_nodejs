Writing a Fetch Request

The fetch API is a way to make HTTP requests in JavaScript. It returns a Promise that resolves with the Response object representing the response to the request. Here's an example of a fetch request:

fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

Getting the Date in JavaScript

In JavaScript, developers can use the Date object to work with dates and times. The Date object provides methods for getting the current date and time, as well as manipulating and formatting date values. Here's an example of creating a new Date object to get the current date and time:

const currentDate = new Date();
console.log(currentDate);
